Dependencies

- Composer dependency manager for PHP
- PHP 7.0 or greater version
- Local server to run PHP script

Setup/Run

- Clone this project from GitHub
- Open a terminal in the root of this project, and run the "composer install" command
- Import the "14oct2020_boldizar_librafire.sql" database file from the SQL folder
- Rename the example.env file in the root of this project to .env and modify the data by your credentials
- Add this project to your web server and load it through your favorite browser
